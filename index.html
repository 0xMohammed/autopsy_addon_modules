The [Autopsy Addon Module Repository](https://github.com/sleuthkit/autopsy_addon_modules) contains information about moduels that can be added to the [Autopsy Digital Forensics Platform](https://www.autopsy.com).

The repository is organized by type of module and then each module has its own folder. You can navigate the repository based on folders or you can search it here for keywords.
